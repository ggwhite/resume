# resume

Use Vue.js to build a resume.

* Node.js: v16.20.2
* npm: v8.19.4



## Project setup

### Install Node.js

```
nvm install v16.20.2 // install Node.js by nvm
```

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
